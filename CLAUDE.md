# Naukri Automation Bot — Claude Context

## Project Overview
Playwright-based bot that automates Naukri profile updates (headline, summary, resume upload) on a daily schedule to boost visibility. Runs on GitHub Actions / Node.js.

## Architecture

```
src/
  auth/
    login.js          # performLogin(), handleOTPIfPresent(), loginWithRetry()
    otpReader.js      # fetchNaukriOTP() — polls Gmail via IMAP for OTP
    sessionManager.js # ensureSession(), saveSessionCookies(), etc.
    encryption.js     # AES-encrypted credential storage
  automation/
    headlineUpdater.js
    summaryUpdater.js
    resumeUploader.js
    delays.js         # randomDelay(), simulateTyping()
  browser/
    browserManager.js
    pageHelpers.js
  utils/
    config.js         # loadAppConfig(), loadSelectors()
    logger.js
    validators.js
config/
  selectors.json      # ALL page selectors live here — class-based first, attr fallbacks
  config.json         # Runtime config (windows, retries, etc.)
  headlines.json / summaries.json
```

## Key Conventions

### Selectors
- **All selectors live in `config/selectors.json`** — never hardcode selectors inside JS files.
- Prefer **CSS class selectors** (e.g. `input#otpField`, `button.loginButton`) over attribute selectors.
- Provide multiple fallbacks ordered from most-specific (class/id) to least-specific (attribute).
- Load via `loadSelectors()` from `src/utils/config.js`.

### OTP Handling
- After password login, `handleOTPIfPresent()` in `login.js` detects the OTP screen.
- OTP selectors are in `config/selectors.json` under `otpForm.otpField` and `otpForm.submitButton`.
- The OTP is fetched automatically from Gmail via IMAP using `fetchNaukriOTP()` in `otpReader.js`.
- Required env vars: `NAUKRI_EMAIL` + `NAUKRI_EMAIL_APP_PASS` (Gmail App Password).
- Polls every 5 seconds, times out after 2 minutes.

### Environment Variables
| Variable | Purpose |
|---|---|
| `NAUKRI_EMAIL` | Naukri login email (also Gmail address for OTP) |
| `NAUKRI_PASSWORD` | Naukri login password |
| `NAUKRI_EMAIL_APP_PASS` | Gmail App Password for reading OTP emails via IMAP |
| `ENCRYPTION_KEY` | AES key for credential encryption (auto-generated by setup) |
| `HEADLESS` | `true`/`false` — browser visibility |

### Adding/Changing Selectors
If a Naukri page update breaks selectors, update **only** `config/selectors.json`. The JS files pick them up automatically via `loadSelectors()`.

## Running
```bash
npm run setup      # First-time: encrypts credentials, generates key
npm start          # Continuous scheduled mode
npm run fast       # Immediate one-off run (uses plain-text env creds)
npm run run-once   # One-off run using encrypted credentials
```
